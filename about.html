Verification of Maximum Power Transfer Theorem

Project Description

This project is a real-time, web-based experiment that verifies the Maximum Power Transfer Theorem in a purely resistive circuit using an ESP32-based system. The theorem states that the power delivered to a load is maximized when the load resistance (RL) equals the internal resistance (RTh) of the source.

Our setup emulates this condition dynamically by varying the load resistance through a digitally controlled potentiometer (MCP41010) and continuously measuring voltage and current using the INA219 sensor module.

Theory

Maximum power is delivered from a source to a load when the load resistance equals the source‚Äôs internal (Thevenin) resistance as seen from the load terminals.

  • At RL = RTh: Power is maximized, efficiency is 50%.
  • RL > RTh: Higher efficiency but less power to load.
  • RL < RTh: More internal loss, less load power.

Note: RL is load resistance, RTh is Thevenin‚Äôs equivalent resistance.
