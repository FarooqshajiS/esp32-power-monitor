Verification of Maximum Power Transfer Theorem

Project Description

This project verifies the Maximum Power Transfer Theorem for purely resistive circuit, which states that maximum power is delivered to the load when the load resistance equals the source resistance.This verification is carried out using a virtua lab simulation,ensuring accurate observation and analysis of the theorem's validity.

Theory

Maximum power is delivered from a source to a load when the load resistance is equal to the source‚Äôs internal (Thevenin) resistance as seen from the load terminals.

  • At RL = RTh: Load power is maximized, but efficiency is only 50% ‚Äî half the power is lost in the internal resistance.
  • If RL > RTh: Efficiency increases, but power delivered to the load decreases.
  • If RL < RTh: More power is dissipated inside the source, and the load receives less power.
    • Note: RL is the load resistance, and RTh is the source resistance (Thevenin‚Äôs equivalent resistance).

    Steps to Find Maximum Power Transferred

    1. Remove the load resistor (RL) from the circuit.
    2. Compute the open-circuit Thevenin voltage (VTh) across the load terminals.
    3. Find the equivalent Thevenin resistance (RTh) by deactivating all independent sources.
    4. Set RL = RTh for maximum power transfer.
    5. Calculate the maximum load current:
      IL,max = VTh / (2 √ó RTh)
    6. Compute the maximum power:
      Pmax = VTh2 / (4 √ó RTh)
